# 🌸 Blooming Desire

### Gaze-based Unconscious Self Exploration Interactive Media Art

**시선 트래킹을 활용한 무의식적 자아 탐구 인터랙티브 미디어 아트 전시**

---

## 📌 Project Overview

**Blooming Desire**는 관람객의 **무의식적인 시선 데이터를 분석**하여

개인의 내면 감정과 욕망을 **꽃이라는 시각적 메타포로 시각화**하는

개인 맞춤형 인터랙티브 미디어 아트 전시 프로젝트입니다.

관람객은 별도의 질문이나 선택 없이 전시에 참여하며,

영상 감상 중 수집된 **시선의 흐름, 응시 시간, 시각적 집중 패턴**을 기반으로

각기 다른 **미디어 아트 결과물**을 경험하게 됩니다.

본 저장소는 전시에서 사용된 **프로그래밍 코드와 함께**,

프로젝트의 **기획 의도, 시스템 구조, 데이터 해석 철학**을 이해할 수 있도록

정리된 프로젝트 아카이브입니다.

---

## 🧠 Background & Motivation

### 왜 ‘시선’인가?

사람의 시선은 의식적인 판단 이전에 **무의식적인 관심과 감정의 흐름**을 드러냅니다.

우리는 스스로를 언어로 설명하지만, 실제로는 말보다 먼저 **시선이 반응**합니다.

본 프로젝트는 다음과 같은 문제의식에서 출발했습니다.

- 기존의 자아 탐구 방식은 **질문·선택·언어 중심**으로 이루어진다
- 이러한 방식은 사회적 맥락과 자기검열의 영향을 받기 쉽다
- 시선 데이터는 보다 **솔직한 내면 반응**을 포착할 수 있다

기존 시선 트래킹 기술은 주로 **UX/UI, 마케팅, 심리 분석**에 활용되어 왔으나,

무의식적 데이터를 **예술적 경험으로 전환하는 시도는 드물었습니다**.

Blooming Desire는 기술과 예술을 결합하여,

**무의식을 ‘분석’이 아닌 ‘경험’으로 마주하게 하는 전시**를 제안합니다.

---

## 🎯 Project Goal

- 시선 트래킹 데이터를 활용한 **무의식 기반 인터랙션 설계**
- 관람객별 **개인 맞춤형 미디어 아트 결과물 생성**
- 정답을 제시하지 않는, **해석 가능한 자아 성찰 경험 제공**
- 기술·예술·심리 데이터 해석이 융합된 **서사형 미디어 아트 구현**

---

## 🌗 Core Concept: Duality of the Self

본 전시는 **자아의 이중성**을 핵심 서사로 삼습니다.

- **추상적 꽃**: 진실된 내면의 자아 (무의식)
- **완벽한 꽃**: 사회적으로 구성된 외면의 자아

관람객의 시선 데이터를 통해 생성된 꽃 이미지는

이 두 자아의 대비를 시각적으로 드러내며,

관람자는 결과를 통해 자신의 내면을 **성찰의 대상**로 마주하게 됩니다.

---

## 🧩 Experience Flow

```
입장 → 체험 → 데이터 분석 → 결과 생성 → 결과 감상
```

1. **Entrance**
    - 암막으로 구성된 어두운 전시 공간 입장
    - 몰입을 유도하는 환경 구성
2. **Gaze-based Experience**
    - 약 90초 분량의 인트로 영상 감상
    - 시선 트래킹 디바이스를 통해 실시간 데이터 수집
3. **Data Analysis**
    - 시각 요소(색상, 위치, 형태)에 대한 시선 집중도 분석
    - 무의식적 감정 및 욕망 해석
4. **Visual Generation**
    - 분석 결과에 따라 꽃의 종류, 색상, 상태, 키워드 결정
    - 개인 맞춤형 시각 이미지 생성
5. **Reflection**
    - 최종 결과물(꽃 이미지)과 함께 해석 문장 제시
    - 데이터를 기반으로 한 분석 리포트 웹 페이지 제공
    - 관람객의 자아 탐구 경험 마무리

---

### 📑 Gaze Data Analysis Report (Post-Exhibition Web Experience)

전시 경험의 마지막 단계에서 관람객은

**자신의 시선 트래킹 데이터를 기반으로 한 분석 리포트 웹 페이지**를 통해

최종 결과물 미디어아트를 **이해하고 해석할 수 있는 설명**을 제공받습니다.

이 리포트는 단순한 결과 요약이 아닌,

“왜 이 색상·형태·움직임·음악·키워드의 작품이 생성되었는지”를

시선 데이터 분석 과정을 통해 단계적으로 설명하는 역할을 합니다.

### ✅Design Intention of the Report

이 데이터 분석 리포트는

시선 데이터를 **과학적으로 ‘설명’하기보다**,

관람자가 자신의 결과를 **이해하고 받아들이는 서사적 장치**로 설계되었습니다.

- 결과는 평가나 진단이 아님
- 데이터는 **‘왜 이런 결과가 나왔는지 납득할 수 있는 근거’**
- 관람자는 자신의 무의식을 **객관적 데이터 + 주관적 해석**으로 동시에 마주함

이를 통해 Blooming Desire는

**경험 → 결과 → 이해 → 성찰**로 이어지는 완결된 전시 구조를 형성합니다.

---

## 🌸 Visual Metaphor: Flower as Emotion

꽃은 본 프로젝트에서 **감정과 욕망을 담는 시각 언어**로 사용됩니다.

- 관람자는 꽃을 **직접 선택하지 않음**
- 시선 데이터 분석 결과에 따라 **자동으로 꽃 유형이 부여**
- 색상, 형태, 움직임은 감정 상태를 은유적으로 표현

> 이 프로젝트의 핵심은
> 
> - *‘선택하지 않았기 때문에 더 나다울 수 있는 결과’**입니다.

## 📊 Data Interpretation Philosophy

Blooming Desire는 시선 데이터를 **진단 도구로 사용하지 않습니다**.

- 결과는 성격 유형이나 심리 테스트가 아님
- 데이터는 하나의 **해석 가능한 단서**
- 관람자는 결과를 통해
    - “이게 나일 수도 있겠다”*라는 **사유의 계기**를 얻는 것이 목적입니다.

기술은 판단자가 아니라,

**자기 성찰을 돕는 매개체**로 작동합니다.

## ⚠️ Limitations & Challenges

- 시선 트래킹 정확도는 조도·환경에 영향을 받음
- 실시간 데이터 전송 시 지연 문제 발생
- 고해상도 생성형 영상의 렌더링 부하

### Solutions

- 캘리브레이션 커스터마이징 및 평균값 기반 보정
- OSC 데이터 필터링 및 비동기 구조 적용
- 일부 장면 사전 렌더링으로 퍼포먼스 최적화

## 🗓️ Exhibition Outcome (June 27–29, 2025)

Blooming Desire는 **2025년 6월 27일부터 29일까지, 총 3일간** 실제 전시 공간에서 운영되었습니다.

전시 기간 동안 관람객은 별도의 조작 없이 전시에 참여했으며,

각 관람 경험은 **실시간 시선 트래킹 데이터 수집 → 분석 → 개인화된 미디어아트 생성**의 흐름으로 완결되었습니다.

전시 기록 영상과 결과물을 통해 확인된 점은 다음과 같습니다.

- 동일한 전시 콘텐츠임에도 불구하고,
    
    관람객마다 **서로 다른 시선 패턴**이 관찰됨
    
- 시선 데이터 차이에 따라
    
    **색감, 형태, 움직임, 내면 키워드가 모두 다른 결과물**이 생성됨
    
- 전시는 관람객 수만큼의 **고유한 작품 결과**를 축적함

전시 종료 후, 관람객은 QR 코드를 통해

**개인별 시선 트래킹 데이터 분석 리포트 웹**에 접근할 수 있었으며,

이를 통해 자신의 결과물이 생성된 이유를 시선 데이터 기반 설명과 함께 확인했습니다.

3일간의 전시 운영을 통해 Blooming Desire는

단일 작품이 아닌, **실제 관람 환경에서 작동하는 데이터 기반 인터랙티브 전시 시스템**으로 검증되었습니다.
